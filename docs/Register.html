What we need from you...

A place to host the sensor

  • The sensor needs to be outsite, ideally on a window sill.
  • It should be in an easy to access, if it needs repair, or maintenance.
  • It will need to be left undisturbed and without interference by anyone as this can affect the readings.
  • A member of the Tech Shed Frome Team will fix it in place for you.

Coronavirus and safety
In order for us to install the air sensor, we will need a member of Tech Shed Frome to come to your premises. We can arrange with you ahead of the visit to ensure social distancing and other measures can be put in place to ensure a safe installation.

Access to power source

  • The device will need a nearby plug to power it, via a flat USB cable (that can fit under a closed window to a plug).
  • The cable is thin enough that many windows will close without disturbing the cable or the device.
  • If you have another type of window, we will assess the best cable for it. Double glazed units are unlikely to be suitable.


How much power does it use?


The sensor runs on 5 volts, the same as a mobile phone. It is powered by a mobile phone charger that needs to stay plugged into the mains and will consume a miniscule amount of power. We estimate the usage to be about 4.161kWh power or about 60 pence per year, based on average UK electricity price.


What if the device stops working?

Technology is great when it works but sometimes it breaks. If the device breaks or stops functioning, we will need to arrange a time to visit the sensor to repair it. We expect the sensors to have a lifespan of at least 1 year but longer. We can arrange this with you at a convenient time.
This is not expected to happen very often if at all.



You must have the following to be an air sensor:

  • A stable wifi connection that is detectable near the window
  • Give access to my property to install the air sensor
  • A window that will host the air sensor and is within 3 meters of a plug
